GetCookin is a React Native mobile application built with Expo. It is designed to help users explore cooking and gastronomy based on their tastes, preferences, and the ingredients they already have.
The goal of the project is to provide a reference point when someone does not know what to eat or what to do with the ingredients available in their refrigerator. Instead of starting from recipes, GetCookin starts from ingredients and user preferences (salty, sweet, acidic, bitter, etc.) to suggest different types of dishes or preparations.
This project is still at an early stage and was developed primarily as a learning project focused on mobile development with React Native and Expo.


## Features
- Free-text ingredient search
- Advanced filtering system:
- Include and exclude ingredients
- Include and exclude tags (salty, sweet, dessert, meal, etc.)
- Category-based filtering
- Browsing a catalog of ingredients and food items
- Animated ingredient cards for improved user experience
- Smart suggestion when no exact match is found


## Tech Stack
- Expo
- React Native
- TypeScript / JavaScript
- Expo Router


## Data and Architecture
- No backend or database at the moment
- Ingredient data (names, tags, categories) is stored locally in the project
- Ingredient images are hosted from external sources
- Nutritional values, calories, and quantities are not implemented yet


## Getting Started

*** Prerequisites *** :

+ Node.js (includes npm)
+ A computer (macOS or Windows)
+ An iOS/Android phone with the Expo Go app installed
+ Project folder


In the terminal :

1. Install dependencies : npm install
2. Start the app : npx expo start


On your phone (Android or iPhone) :

1. Install Expo Go (App Store / Google Play)
2. Scan the QR code shown by Expo
    Android : scan directly in Expo Go
    iPhone : scan with the Camera app, then open in Expo Go
3. Start exploring the app ;)

**** If it bugs : npx expo start -c ****

-	Basic UI and early-stage design
-	The application currently runs only through Expo Go and is not published on app stores
